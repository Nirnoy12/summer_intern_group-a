# How to Contribute

1. **Fork** the repository to your own GitHub account.
2. **Clone** your fork locally.
3. **Sync** your fork with the upstream main branch before starting work.
4. **Create a branch** (`feature/backend-auth`, `fix/ui-bug`).
5. **Commit** your changes.
6. **Push** to your fork.
7. Open a **Pull Request (PR)** against the `main` branch of the upstream repository.

**Folder Structure:**
- `/frontend`: React + Shadcn UI (Run `npm run dev`)
- `/backend`: FastAPI + SQLModel (Run `uvicorn main:app --reload`)
